  • queryRegistrationResult(of:completionHandle:)

    Polling device pairing result.

    Since device pairing is an asynchronous process, the time to successfully enter the network varies depending on the network quality, so developers need to poll the interface until the device list is queried or stop polling when the timeout expires.

    The polling period is once per second, and the timeout is recommended to be 100 seconds.
